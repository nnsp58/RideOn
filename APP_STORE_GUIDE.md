# 📱 App Store Publishing Guide

## Options Available

### 1. Google Play Store (Recommended for India)
- **Cost:** $25 (one-time)
- **Requirements:**
  - Google Developer account
  - App icon (512x512)
  - Screenshots (phone screenshots)
  - Privacy Policy URL
  - Release APK/AAB

**Steps:**
1. Go to [play.google.com/console](https://play.google.com/console)
2. Create app → Fill details
3. Upload AAB file
4. Fill store listing
5. Submit for review

### 2. Direct Share (Family & Friends)
- Debug APK bhejo WhatsApp/Email se
- APK install karke use karenge
- No review needed
