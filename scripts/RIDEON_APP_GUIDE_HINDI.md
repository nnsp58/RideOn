# RideOn App - Poora Guide Hindi Mein

## "RideOn hai kya? Bilkul aasan bhasha mein samjhein"

---

## 1. RideOn Kya Hai?

RideOn ek car-pooling app hai. Matlab agar aapko kahan jaana hai aur kisi ka raasta wahan jaane ka hai - toh aap ek saath chal sakte ho. Aapka kharcha bhi kam hoga, driver ko earning milegi, aur gaadiyan kam chalen-gi - sabko fayda!

**Ek line mein:** "Share karo, save karo."

---

## 2. App Kaise Shuru Karein?

### Pehla Step - Sign Up

1. App kholo
2. Phone number daalo
3. Aapke phone par ek OTP code aayega (SMS ke through)
4. OTP daalo - account ready hai!

### Email/Password Se Bhi Login Ho Sakta Hai

- Email aur password se bhi account bana sakte ho
- Ya pehle se bana account login kar sakte ho

### Phir Kya Hoga?

- **Profile Setup:** Apna naam, photo, aur bio add karo
- **Vehicle Setup:** Agar gaadi hai toh model, number plate, color aur type (Car/Bike/Van) daalo

> **Note:** Vehicle daalna optional hai. Sirf passenger bhi ban sakte ho bina gaadi add kiye. Lekin ride publish karne ke liye vehicle aur document verification zaroori hai.

---

## 3. Ride Kaise Dhoondhein Aur Book Karein?

### Step 1: Search Karo

1. Home screen par "Search Rides" button dabao
2. "Kahan se?" (From) likho ya map par pin karo
3. "Kahan tak?" (To) likho ya map par pin karo
4. Date chuno - aaj kal ka din ya future ka
5. "Search" dabao

**Kaise Kaam Karta Hai:**
RideOn smart hai - aapke raaste ke aas-paas (80 km tak) guzarne wali rides dhoondh ke lata hai. Agar koi Delhi se Jaipur ja raha hai aur aapko Gurgaon se Jaipur jaana hai - RideOn dhundh lega!

### Step 2: Ride Chuno

- Jo rides aayengi, usme dekho:
  - Driver ka naam aur rating
  - Kahan se kahan ja raha hai
  - Kitna time nikal raha hai
  - Kitne seats khali hain
  - Price (daal)

### Step 3: Book Karo

1. Ride par click karo - details kholengi
2. Sab dekho - route map, driver info, time
3. "Book Ride" dabao
4. Confirm karo
5. Done! Booking confirm ho gayi

**Baare Mein Kuch Zaroori:**
- Price automatic calculate hota hai - jitna aapka raasta utna charge
- Minimum ₹30 (kitna bhi short journey ho)
- Booking departure time se 5 minute pehle tak ho sakti hai - uske baad nahi
- Agar last seat already book ho gayi toh ride "Full" dikhayi degi

---

## 4. Apni Ride Kaise Banayein? (Driver)

> **Zaroori:** Driver banne ke liye vehicle add karna aur document verification approve hona zaroori hai.

### Ride Publish Karo:

1. App par "Publish Ride" button dabao (ya + button)
2. Batao:
   - **Kahan se jaana hai** (Start point)
   - **Kahan tak** (End point)
   - **Kab jaana hai** (Date aur Time)
   - **Kitni seats hai** (1, 2, 3... total seats)
   - **Price per seat** (aap decide karo)
3. Optional:
   - Route ke beech mein kuch stops add karo (map par tap karke)
   - Description likho (jaise "AC Car", "Highway se jaunga")
4. Ride rules chuno:
   - No Smoking - Dhoomrapaan mana hai
   - No Music - Music bajana hai ya nahi
   - No Heavy Luggage - Bhaari saamaan nahi
   - No Pets - Jaanwar nahi chalenge
   - Negotiation - Price baatcheet ho sakti hai
5. "Publish" dabao

**Baare Mein Kuch Zaroori:**

- Jab tak koi nahi book karega - aapki ride dikhegi jab bhi koi search karega
- Jab koi book karega - uske baare mein ek notification milega
- Agar aapki ride us route par hai jise kisi ne search kiya tha - toh usko notification jaayega ki "Ek nayi ride available hai!"
- Aap ride cancel kar sakte ho - lekin saare passengers ko notification chala jaayega

---

## 5. Booking Cancel Kaise Karein?

### Passenger Ke Liye:

1. "My Bookings" par jao
2. Apni booking kholo
3. "Cancel" dabao
4. Reason likho (optional)
5. Confirm karo

Kya hoga:
- Booking cancel ho jaayegi
- Jo seat book thi woh wapas available ho jaayegi
- Driver ko notification chala jaayega

### Driver Ke Liye:

Driver apne passengers ki koi bhi booking cancel kar sakta hai:
1. "My Rides" par jao
2. Ride select karo
3. Passenger list mein cancel button dabao
4. Ya poora ride cancel karo - saari bookings ek saath cancel

---

## 6. Chat Kaise Karein? (Driver aur Passenger)

Aapas mein seedha baat kar sakte ho - bina phone number share kiye:

1. Kisi bhi active booking par "Chat" button dabao
2. Message likho aur send karo
3. Samne waala reply karega - real-time (turant)
4. Pickup point, late hona, ya koi baat - sab chat mein ho jaayega

**Extra Features:**
- Message history rehti hai
- Naya message aate hi notification aayega
- Bina app khole bhi push notification milega

---

## 7. Profile Kaise Edit Karein?

1. Profile tab par jao
2. Photo change, naam change, bio update - sab possible hai
3. "Save" dabao

**Document Upload:**
- Driving License (aage aur peeche photo dikhani hogi)
- Gaadi ka RC (Registration Certificate)
- ID Proof (Aadhar, PAN etc.)
- PUC Number
- Insurance Number

Admin aapke documents check karega. Jab tak approve nahi hota, tab tak ride publish nahi kar paoge. Yeh security feature hai - taaki sab passengers ko bharosa ho.

---

## 8. Ratings Aur Reviews

Har ride ke baad rating de sakte ho:
- 1 se 5 stars tak
- Optional comment bhi likh sakte ho

- Achhe drivers ki rating badhti hai
- Achhe passengers ko drivers pasand karenge
- Rating dekh ke dono decide kar sakte hain ki saath jaana hai ya nahi

---

## 9. SOS Button - Emergency Help

Agar safar mein kuch galat lage ya emergency hai:

1. Har screen par ek **SOS** button dikhayega
2. Button dabao
3. GPS location automatic jaayega admin ko
4. Admin turant dekh lega

Admin ko yeh dikhega:
- Aapka naam
- Aapki exact location (kahan ho aap - yeh GPS se pata chalta hai)
- Emergency type (general ya custom)
- Kab trigger hua

Admin SOS resolve kar sakta hai - lekin tum khud bhi cancel kar sakte ho agar sab theek hai.

## 16. Permissions - Kyun Chahiye?

| Permission | Kyun | Kya Nahi Karte |
|------------|------|----------------|
| Location (GPS) | Map, ride search, SOS | Kabhi nahi track karein bina aapke jaankari ke |
| Camera | Profile photo, document upload | Kabhi bina permission photo nahi lenge |
| Internet | Rides search, chat, notifications | Aapka data secure hai - encrypt hai |

---

## Permissions Detail - Transparent Aur Clear

Hum aapko sab bata dete hain:

### Location Permission
**Kyun chahiye:** Map par aapko dikhane ke liye, rides search karne ke liye, aur emergency mein SOS location bhej ke liye

**Kya karte hain:** Sirf app use karte time location lete hain

**Kya nahi karte:** Background mein track nahi karte. Location ka data kisi ko nahi bechte.

### Camera Permission
**Kyun chahiye:** Profile photo upload karne ke liye, aur driving license/gaadi papers ke photos lene ke liye

**Kya karte hain:** Jab aap photo lena chahein, tabhi camera open hoga

**Kya nahi karte:** Bina permission camera access nahi karte. Photo bina aapke kahe nahi lete.

### Internet Permission
**Kyun chahiye:** Rides search karne ke liye, chat karne ke liye, notifications lene ke liye

**Kya karte hain:** Sirf app ke kaam ke liye internet use karte hain

**Kya nahi karte:** Aapka personal data kisi aur company ko nahi bechte. Data secure aur encrypt rehta hai.

## 17. Common Questions (FAQ)

**Q: RideOn free hai?**
A: Haan, app download karna aur use karna bilkul free hai. Aap sirf ride ka paisa driver ko dete ho.

**Q: Main sirf passenger ban sakta hoon?**
A: Haan! Vehicle add kiye bina bhi ride book kar sakte ho. Ride publish karne ke liye vehicle + verified documents chahiye.

**Q: Kya cash mein payment hota hai?**
A: Abhi Payment driver aur passenger ke beech direct hota hai - cash, UPI, ya kisi aur tarah se. App ke andar abhi payment gateway nahi hai.

**Q: Agar driver nahi aaya toh?**
A: Driver cancel kar sakta hai - aapko notification milega. Aap doosri ride book kar sakte ho.

**Q: Rating kyun zaroori hai?**
A: Rating se bharosa banta hai. Achhe drivers ko zyada passengers milte hain, achhe passengers ko drivers khushi se le jaate hain.

**Q: Kya main ek hi ride mein ek se zyada seat book kar sakta hoon?**
A: Haan - jitni seats available hon utni book kar sakte ho.

**Q: Ride cancel karne par paise wapas?**
A: Payment seedha driver ko hota hai - toh cancellation refund driver se discuss karna padega.

**Q: Mera data safe hai?**
A: Haan. Hum sirf zaroori data lete hain jo app chalane ke liye chahiye. Data nahi bechte. Supabase secure cloud par data store hota hai.

**Q: Agar mera khoya toh?**
A: Password reset kar sakte ho. Email se password change kar sakte ho. Account recover kar sakte ho.

**Q: RideOn kaunse areas mein kaam karta hai?**
A: Puri India mein kaam karta hai. Jahan bhi rides available hon wahan. Abhi shuruwat hai - jitna zyada log use karenge, utni zyada rides milengi.

## 19. Problem Aaye Toh?

| Problem | Solution |
|---------|----------|
| App login nahi ho raha | Password sahi check karo, ya "Forgot Password" try karo |
| OTP nahi aa raha | Thoda wait karo, "Resend OTP" dabao, network check karo |
| Location nahi aa rahi | GPS on karo, location permission do, app restart karo |
| Ride book nahi ho rahi | Seats full honge, past time hogi, network issue hoga |
| Chat nahi aa rahi | Internet check karo, kaun si ride nahi dikh raha - usko doosri ride hai kya? |
| Push notifications nahi aa rahe | App settings mein notifications on karo, OneSignal check kar |
| App crash ho raha hai | App update karo, cache clear karo, phir try karo |

**Email karo:** support@rideon.com
